TECHNICAL DETAILS
================================================================================

Languages Supported: Indonesian (ID), English (EN), Mandarin Chinese (ZH)
Default Language: Indonesian (ID)
Fallback Behavior: Always falls back to Indonesian if translation missing

Static UI Translation:
  - Location: static/locales/{id,en,zh}.json
  - Usage: data-i18n="prodi.register_now"
  - Handler: i18n-init.js
  - Keys: 15 prodi-specific keys

Dynamic Content Translation:
  - Location: ContentBlock JSON field in database
  - Structure: {"id": "...", "en": "...", "zh": "..."}
  - Usage: data-i18n-content="hero_section.title"
  - Handler: i18n-content.js (NEW)

Integration:
  - i18n-init.js fires languageChanged event
  - i18n-content.js listens and updates dynamic content
  - Template filters handle server-side rendering
  - Client-side JS handles dynamic switching
